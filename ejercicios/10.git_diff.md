# Ejercicio 10: Ver Cambios entre Commits

Aprendemos a observar las diferencias entre commits usando `git diff`.

---

1. Haz al menos dos commits con cambios en un mismo archivo.

2. Usa `git diff` para comparar los cambios entre los commits.

3. Usa `git log` para identificar los hashes de los commits y compara directamente dos versiones específicas con `git diff <commit1> <commit2>`.

4. Observa las diferencias y reflexiona sobre cómo el comando `git diff` puede ayudarte a depurar código.
