# Ejercicio 03

El objetivo es crear una rama en nuestro repositorio y comprobar que es distinta a la rama original `main`.

Comando:

```
git checkout <rama>
```

---


**( en vuestro repositorio )**

1. Creamos un archivo cualquiera y hacemos un commit

2. Creamos otra rama, por ejemplo `mi-rama`

3. Hacemos un cambio diferente

4. Comprobamos que los historiales difieren con `git log` y `git checkout`

5. Hacemos una captura del resultado
