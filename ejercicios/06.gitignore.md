# Ejercicio 06: Ignorar Archivos con .gitignore

Ignorar archivos usando el `.gitignore` en el repositorio.

---

1. Crea un archivo .gitignore en tu repositorio.

2. Añade una regla para ignorar un archivo o carpeta (por ejemplo, *.log o node_modules/).

3. Intenta añadir un archivo que coincida con las reglas de .gitignore y verifica que no se añade al staging.

4. Comprueba el contenido de .gitignore con el comando git status.
