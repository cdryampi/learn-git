# Ejercicio 09: Etiquetas (Tags)

Aprendemos a usar tags con `git tag`.

---

1. Crea un nuevo commit en tu repositorio.

2. Crea una etiqueta ligera (lightweight) para el commit con `git tag <nombre-de-la-etiqueta>`.

3. Crea una etiqueta anotada (annotated) para otro commit con `git tag -a <nombre-de-la-etiqueta> -m "Descripción"`.

4. Lista las etiquetas existentes con `git tag`.

5. **(opcional)** Sube las etiquetas al repositorio remoto con `git push origin --tags` (si tiene un repositorio asociado en remoto).
